CS50P is Harvard’s free introductory Python course, officially titled “CS50’s Introduction to Programming with Python.”

It’s designed for beginners, including people with no prior programming experience, and focuses entirely on Python rather than general computer science.

What it covers:

  -Python basics: variables, types, conditionals, loops, functions

  -Program structure: arguments, return values, scope

  -Working with data: lists, dictionaries, files, libraries

  -Problem solving: exceptions, debugging, regular expressions

  -More advanced fundamentals: object-oriented programming, classes, properties, unit testing
